Css selectors

ID селектори- самий потужний тип селекторів. Це означає, що вони "витісняють" інші типи селекторів, а стилі вказані з цим селектором застосовуються першочергово.

Селектор класу - самий використовуваний і універсальний тип селектора.

Тег класу - тип селекторів дуже корисний при зміні унікальних властивостей HTML документа.

Селектор атрибуту - ви можете заявити, що атрибутні селектори більш корисні ніж селектори класу, тому що вони мають однакову важливість, але можуть бути будь-яким атрибутом, не тільки class, а також ми можемо вибирати їх за значенням.

Селектор позиції - ви можете обрати елементи за їх положенням в HTML.
